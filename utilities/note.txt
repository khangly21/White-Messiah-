## Code mẫu fix

```python
from transformers import pipeline
import PyPDF2

summarizer = pipeline(
    "summarization",
    model="facebook/bart-large-cnn"
)

def extract_text_from_pdf(pdf_path):
    text = ""

    with open(pdf_path, "rb") as f:
        reader = PyPDF2.PdfReader(f)

        for page in reader.pages:
            page_text = page.extract_text()
            if page_text:
                text += page_text + "\n"

    return text


def chunk_text(text, max_words=500):
    words = text.split()
    chunks = []

    for i in range(0, len(words), max_words):
        chunk = " ".join(words[i:i + max_words])
        chunks.append(chunk)

    return chunks


def summarize_pdf(pdf_path, max_len=200, min_len=50):
    text = extract_text_from_pdf(pdf_path)

    if not text.strip():
        return "Không đọc được text từ PDF. Có thể PDF là ảnh scan, cần OCR."

    chunks = chunk_text(text, max_words=500)

    partial_summaries = []

    for chunk in chunks:
        if len(chunk.split()) < 50:
            continue

        result = summarizer(
            chunk,
            max_length=max_len,
            min_length=min_len,
            do_sample=False,
            truncation=True
        )

        partial_summaries.append(result[0]["summary_text"])

    combined_summary = " ".join(partial_summaries)

    if len(combined_summary.split()) > 500:
        final_result = summarizer(
            combined_summary,
            max_length=max_len,
            min_length=min_len,
            do_sample=False,
            truncation=True
        )
        return final_result[0]["summary_text"]

    return combined_summary
```

chúng có tác dụng gì trong trang web:

COPI:
Hai thư viện đó có tác dụng bổ sung cho trường hợp **PDF không phải text mà là bản scan hình ảnh** (ví dụ whitepaper được lưu dưới dạng ảnh chụp):

- **pdf2image**  
  👉 Dùng để **chuyển từng trang PDF thành ảnh** (PNG/JPEG). Vì PyPDF2 chỉ đọc được text, nếu PDF là scan thì PyPDF2 sẽ trả về `None`. Lúc này pdf2image giúp bạn có ảnh của từng trang để xử lý tiếp.

- **pytesseract**  
  👉 Đây là wrapper của Tesseract OCR. Nó sẽ **nhận ảnh từ pdf2image và nhận diện chữ trong ảnh** → trả về text. Nhờ đó bạn có thể tóm tắt cả những whitepaper scan, không chỉ PDF text.

---

## 📊 Quy trình trong web app
1. Người dùng upload PDF.  
2. Nếu PyPDF2 đọc được text → tóm tắt như bình thường.  
3. Nếu PyPDF2 không đọc được (PDF scan) → dùng pdf2image chuyển trang thành ảnh.  
4. Dùng pytesseract OCR để lấy text từ ảnh.  
5. Đưa text này vào pipeline tóm tắt.  

---

## 🎯 Kết luận
- **PyPDF2**: đọc PDF text.  
- **pdf2image + pytesseract**: fallback cho PDF scan (chỉ có hình ảnh).  
- Nhờ đó app của bạn có thể xử lý **mọi loại whitepaper**, kể cả bản scan.  

👉 Bạn có muốn mình viết luôn một hàm `summarize_pdf_with_ocr` kết hợp PyPDF2 và OCR fallback, để app tự động chọn phương pháp phù hợp không?
